## 🚀 Getting Started (Quick Setup for the Team)

Follow these steps **in order** the first time you set up the project on your machine.

### Prerequisites

Make sure you have the following installed before continuing:

- **Node.js** (v18 or higher) — download from [https://nodejs.org](https://nodejs.org)
  - To verify: open a terminal and run `node -v`
- **npm** (comes with Node.js)
  - To verify: run `npm -v`

### Step 1 — Navigate to the Angular project folder

```bash
cd angular-project/find-my-room-app
```

### Step 2 — Install dependencies

This only needs to be done **once** (or after pulling new changes that modify `package.json`):

```bash
npm install
```

This will download all the required libraries into a `node_modules/` folder.

### Step 3 — Start the development server

```bash
npm start
```

Once it finishes compiling, open your browser and go to:

**[http://localhost:4200](http://localhost:4200)**

The app will automatically reload whenever you save a file.

### Step 4 — Stop the server

Press `Ctrl + C` in the terminal.
